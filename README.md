## Run Locally ğŸ’»

```
> Clone the repo
    >> For Windows: Git Bash
    >> For Linux: Terminal
    >> git clone https://github.com/aditya-190/a2zdsa.git
    
> Change the folder location to your source Code Folder
    >> cd a2zdsa/src
    
> Install the dependencies
    >> yarn install
    
> Start the server
    >> yarn dev
    
> a2zdsa runs on port 3000 of your Local Machine
```
